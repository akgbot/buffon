Randomness metrics

Each metric is color-coded by how closely it matches the expected value for a truly random sequence.

ColorMeaning
■ GreenLooks random
■ TealSlightly uniform
■ BlueSuspiciously uniform — characteristic of quasi-random sequences like Halton
■ YellowMild clustering or correlation
■ RedStrong non-random structure

Spatial & Angle χ²/df

Chi-squared divided by degrees of freedom. For random data the mean is 1.0 and std dev is √(2/df) — roughly ±0.18 for the 8×8 grid (df=63). Thresholds: blue <0.3, teal 0.3–0.5, green 0.5–1.5, yellow 1.5–2.5, red >2.5.

Serial autocorrelation

Lag-1 Pearson correlation of the last 500 crossing results. The significance threshold for zero correlation is roughly 1/√500 ≈ 0.045. Thresholds: green |r|<0.05, yellow 0.05–0.12, red >0.12.

How it works

A needle of length L dropped on a floor with lines spaced D apart crosses a line with probability:

P = 2L / (π · D)

Rearranging to estimate π:

π ≈ 2L · N / (D · C)

where N = total drops, C = crossings
